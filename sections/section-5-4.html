空氣落菌檢測方法：落下菌法 vs 空氣採樣法

了解兩種常用方法的原理與操作，選對工具更精準掌握環境微生物濃度；本區作為案例成效前的背景知識補充。

落下菌法 Settle Plate

Passive

利用自然沉降收集空氣中微生物，操作簡單、低成本，適合巡檢趨勢觀察。

  • 將已凝固之培養基暴露於空氣一段時間，落菌自然沉降。
  • 監測總生菌數；不需儀器、流程直覺。
  • 結果為菌落數（可換算 CFU/m² / 時間）。
巡檢適用：實驗室 / 食品廠 等日常環境

空氣採樣法 Active Sampling

Active

以設備主動抽取固定體積空氣並將微生物捕集於培養基表面，數據可量化比較。

  • 以採樣器控制流量與體積，提升一致性。
  • 可直接換算 CFU/m³，利標準對照。
  • 適用高潔淨 / 醫療或需精確趨勢的場域。
精確量化：潔淨室 / 醫療空間 等關鍵場域

操作流程（精簡）

落下菌法

  1. 選擇適當培養基
  2. 按設定時間暴露
  3. 細菌 30–35°C / 真菌 25–30°C 培養
  4. 計數菌落並紀錄條件

空氣採樣法

  1. 校正採樣器流量
  2. 依高度與點位採樣
  3. 培養與計數換算 CFU/m³
  4. 完整紀錄流量、體積、環境

＊條件依法規 / 內控標準調整；此為常見範圍示意。

方法比較

項目 落下菌法 空氣採樣法
量測指標菌落數（可換算面積/時間）CFU/m³
設備/成本培養皿 + 培養箱，低需採樣器，較高
精確度受氣流/沉降影響體積可控，可比較
適用場景日常巡檢高潔淨或醫療

應用場景

實驗室 食品廠 醫療場所 潔淨室 室內空氣評估 黴菌趨勢

注意事項

採樣與設備

  • 採樣前校正流量
  • 操作與器材保持無菌

培養條件

  • 細菌 30–35°C
  • 真菌 25–30°C

紀錄與判讀

  • 完整紀錄條件與結果
  • 對照歷史趨勢與標準

本區說明落下菌法與空氣採樣法的概念、流程、比較與應用場景，協助選擇合適的環境微生物監測方法。
